# ecommerce-platform
The E-Commerce Website is an online shopping platform that allows users to browse products, view product details, add items to a shopping cart, and place orders conveniently. The project is designed to provide a simple, secure, and user-friendly shopping experience.
An admin panel is provided to manage products, update product information, view customer orders, and maintain the website database.

Technologies Used
Frontend: HTML, CSS, JavaScript, Bootstrap
Backend: PHP
Database: MySQL
Server: XAMPP
